# NIFTY Options Paper Trader (UI Demo)

This is a light-themed demo UI for a paper trading dashboard built with React, TypeScript, Vite and Tailwind CSS. It contains four independent trading workspaces stacked vertically and uses mock data only.

Getting started

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

Notes

- No backend or APIs are included.
- UI is ready to integrate charting libraries and live market data.
